# Contributing to webaudits.org

Contributions are welcome to help improve the project.

### How to contribute

* **Reporting Issues:** If a bug is found or a feature is requested, please [open an issue](https://github.com/wenzf/webaudits/issues).
* **Pull Requests:** 1. Fork the repository.
    2. Create a feature branch.
    3. Submit a Pull Request with a clear description of the changes.

### Areas for Contribution

Assistance is particularly appreciated in **code optimization**, **methodology improvements**, **translations**, and **documentation**.

---
*By contributing, the contributor agrees that the code will be licensed under the project's existing Open Source license.*
